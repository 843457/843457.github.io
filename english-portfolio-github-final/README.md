# English Class Portfolio

## Setup Instructions

1. **Create a folder** named `english-portfolio` in your GitHub Pages repository root
2. **Copy all files** from this export into that folder:
   - `index.html`
   - `assets/` folder (CSS and JS files)
   - `images/` folder (background images)
   - `README.md` (this file)

3. **Update the links** in `index.html`:
   - Find the three `href="https://docs.google.com/..."` links
   - Replace with your actual Google Docs and Google Slides URLs
   - The three links are for:
     - Evidence #1: Course Resource (Google Doc)
     - Evidence #2: Slideshow Presentation (Google Slides)
     - Evidence #3: SOP Writing (Google Doc)

4. **Update the images** (optional):
   - Replace the images in the `images/` folder with your own screenshots
   - The three images are:
     - `hero-background.png` - Hero section background
     - `evidence-accent.png` - Evidence #1 and #3 linked images
     - `growth-illustration.png` - Evidence #2 slideshow image

5. **Push to GitHub** - your site will be live at `https://kp344.github.io/english-portfolio/`

## Customization

### Changing Student Information
Edit the student info in `index.html`:
- Search for "Kunj Patel" and replace with student name
- Search for "843457" and replace with student number
- Search for "EN1W0-6" and replace with course code

### Changing Google Doc/Slides Links
Find these lines in `index.html` and replace the URLs:
- `href="https://docs.google.com/document/d/YOUR_DOCUMENT_ID/edit"` (Evidence #1)
- `href="https://docs.google.com/presentation/d/YOUR_PRESENTATION_ID/edit"` (Evidence #2)
- `href="https://docs.google.com/document/d/YOUR_SOP_DOCUMENT_ID/edit"` (Evidence #3)

### Changing Images
Replace the image files in the `images/` folder with your own screenshots:
- Slideshow screenshots should be landscape format
- Google Doc screenshots should be portrait format
- Recommended size: 800x600px or larger
